# Java 26 Reflection API POC

This POC explores the Java Reflection API to inspect and manipulate classes at runtime. It shows how to retrieve class metadata, access and modify fields (even private ones), and invoke methods dynamically.

- [Class Metadata (Name, Simple Name)](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/src/main/java/com/guisfco/Main.java#L9)
- [Listing Fields, Methods, and Constructors](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/src/main/java/com/guisfco/Main.java#L43)
- [Accessing and Reading Private Fields](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/src/main/java/com/guisfco/Main.java#L64)
- [Invoking Private Methods](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/src/main/java/com/guisfco/Main.java#L76)
- [Modifying Field Values](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/src/main/java/com/guisfco/Main.java#L83)
