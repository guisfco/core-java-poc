# Java 26 Streams and Collections POC

This POC covers the basics of Java Collections (Lists, Maps) and Arrays, focusing on the differences between immutable and modifiable structures. It also demonstrates how to use Streams with arrays and maps, and how to convert primitive arrays to object-oriented lists.
