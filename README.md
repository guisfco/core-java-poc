# POCs

A collection of Proof of Concepts (POCs) exploring various features, APIs, and languages. Each project focuses on a specific area, demonstrating practical usage and implementation details.

## Projects

- [Functional Interfaces (Java)](https://github.com/guisfco/lc-pocs/blob/main/java-26-functional-interfaces/): Exploring built-in and custom functional interfaces with practical examples like payment pipelines and retry mechanisms.
- [Properties API (Java)](https://github.com/guisfco/lc-pocs/blob/main/java-26-properties-api/): Demonstrating the management of configuration using the `java.util.Properties` API, including XML support.
- [Reflection API (Java)](https://github.com/guisfco/lc-pocs/blob/main/java-26-reflection-api/): Investigating runtime class inspection, field manipulation, and dynamic method invocation.
- [Streams and Collections (Java)](https://github.com/guisfco/lc-pocs/blob/main/java-26-streams-1/): Covering the basics of Collections, Arrays, and Streams, including immutability and primitive handling.
- [Streams and Functional Interfaces Part 2 (Java)](https://github.com/guisfco/lc-pocs/blob/main/java-26-streams-2/): Advanced stream operations using specialized functional interfaces like Supplier, Consumer, and Operators.

Each project contains its own `README.md` with detailed topic links to the source code.
